Flight Checklists

This document contains all standardized checklists for PartyBus VA operations. All checklists must be completed in the order specified. Check items are to be read aloud and verified before proceeding to the next item.

Important: Checklists are mandatory for all flights. Deviation from checklists may only occur in emergency situations. All items must be verified before proceeding.
Pre-Flight Inspection Checklist

Purpose: To be performed before every flight to ensure aircraft is airworthy and properly configured.

Location: External walkaround and cockpit

Time Required: Approximately 10-15 minutes


Exterior Inspection:

□ Nose landing gear - tires, strut, doors, lights
□ Left main landing gear - tires, strut, doors, lights
□ Right main landing gear - tires, strut, doors, lights
□ Fuselage - no visible damage, panel security
□ Left wing - leading edge, slats, flaps, spoilers, lights
□ Right wing - leading edge, slats, flaps, spoilers, lights
□ Horizontal stabilizer - no damage, control surfaces free
□ Vertical stabilizer - no damage, rudder free
□ Engines - cowlings secure, no visible fluid leaks, intake/exhaust clear
□ Static ports - clear and unobstructed
□ Pitot tubes - clear and unobstructed
□ Radome - secure, no damage
□ Fuel quantity - verify visually and compare with fuel gauges
□ External lights - navigation, strobe, landing, taxi lights functional
□ Emergency equipment - location verified (if applicable)
Cockpit Pre-Flight Checklist

Purpose: To ensure cockpit systems are properly configured and functional before flight.

Location: Cockpit


□ Parking brake - SET
□ Battery switches - ON (if applicable)
□ External power - CONNECTED (if available)
□ Avionics master - ON
□ Flight instruments - CHECK readings
□ Navigation displays - CHECK functionality
□ Radio frequencies - SET and VERIFIED
□ Transponder - SET to standby (1200 when airborne)
□ Fuel quantity - VERIFIED against flight plan requirements
□ Fuel selectors - CHECK position
□ Circuit breakers - ALL IN
□ Engine fire extinguisher - ARMED (if applicable)
□ Emergency equipment - LOCATION VERIFIED
□ Documents - AIRCRAFT MANUAL, CHARTS, FLIGHT PLAN available
□ ACARS system - STARTED and CONNECTED
□ Flight plan - LOADED and VERIFIED
Engine Start Checklist

Purpose: Standard procedure for starting aircraft engines.

Location: Cockpit


□ Parking brake - SET
□ Fuel pumps - ON
□ Throttles - IDLE
□ Fuel mixture - RICH (if applicable)
□ Master switch - ON
□ Ignition - ON
□ Starter - ENGAGE
□ Oil pressure - CHECK rising (within 30 seconds)
□ Engine RPM - CHECK stable
□ Engine temperature - CHECK normal range
□ Electrical load - CHECK ammeter
□ Engine instruments - ALL IN GREEN
□ Repeat for second engine (if applicable)
Before Taxi Checklist

Purpose: Final checks before leaving parking area.


□ Flight controls - FREE and CORRECT movement
□ Flaps - SET for takeoff
□ Trim - SET for takeoff
□ Parking brake - RELEASE
□ Taxi lights - ON
□ Navigation lights - ON
□ Transponder - STANDBY
□ Radio - SET to ground control frequency
□ Clearance - OBTAINED and READ BACK
□ Taxi instructions - UNDERSTOOD
Before Takeoff Checklist

Purpose: Final verification before takeoff.


□ Runway - VERIFIED correct
□ Wind - CHECK direction and velocity
□ Takeoff clearance - OBTAINED
□ Flaps - SET for takeoff
□ Trim - SET for takeoff
□ Fuel quantity - VERIFIED adequate
□ Engine instruments - ALL IN GREEN
□ Flight instruments - CHECKED
□ Navigation - SET and VERIFIED
□ Transponder - ON and SET
□ Landing gear - DOWN and LOCKED
□ Lights - AS REQUIRED
□ Doors - SECURED
□ Cabin - SECURED (if applicable)
□ Takeoff briefing - COMPLETED
After Takeoff Checklist

Purpose: Post-takeoff configuration changes.


□ Positive rate - VERIFIED
□ Landing gear - RETRACT
□ Landing gear lights - ALL OUT
□ Flaps - RETRACT to UP
□ Climb power - SET
□ Climb speed - ESTABLISH
□ Navigation - VERIFY tracking
□ Radio - CONTACT departure control
□ Engine instruments - MONITOR
□ Cabin altitude - CHECK (if pressurized)
Cruise Checklist

Purpose: Configuration for cruise flight.


□ Cruise altitude - ATTAINED
□ Cruise power - SET
□ Cruise speed - ESTABLISH
□ Autopilot - ENGAGE (if desired)
□ Navigation - VERIFY on course
□ Fuel flow - CHECK
□ Engine instruments - ALL IN GREEN
□ Weather - MONITOR
□ Traffic - MONITOR
□ Radio frequencies - MONITOR
□ ACARS - VERIFY tracking
Descent Checklist

Purpose: Preparation for descent and approach.


□ Top of descent - CALCULATED
□ Approach briefing - COMPLETED
□ Approach charts - REVIEWED
□ Landing data - CALCULATED
□ Fuel quantity - VERIFIED adequate
□ Navigation - SET for approach
□ Radio - CONTACT approach control
□ Altimeters - SET to local QNH
□ Autopilot - DISENGAGE (if applicable)
□ Descent checklist - COMPLETE
Before Landing Checklist

Purpose: Final configuration for landing.


□ Landing clearance - OBTAINED
□ Runway - VERIFIED correct
□ Wind - CHECK direction and velocity
□ Landing gear - DOWN and LOCKED
□ Landing gear lights - ALL GREEN
□ Flaps - SET for landing
□ Trim - SET for landing
□ Approach speed - ESTABLISH
□ Landing lights - ON
□ Navigation - VERIFY approach
□ Engine instruments - ALL IN GREEN
□ Final approach - STABILIZED
After Landing Checklist

Purpose: Post-landing procedures.


□ Reverse thrust - ENGAGE (if applicable)
□ Brakes - APPLY as needed
□ Reverse thrust - DISENGAGE
□ Flaps - RETRACT
□ Speed brakes - RETRACT
□ Landing lights - OFF
□ Transponder - STANDBY
□ Radio - CONTACT ground control
□ Taxi instructions - OBTAIN
Shutdown Checklist

Purpose: Engine shutdown and post-flight procedures.


□ Parking brake - SET
□ Engines - IDLE for 1-2 minutes (cooling)
□ Fuel pumps - OFF
□ Throttles - IDLE
□ Mixture - IDLE CUTOFF (if applicable)
□ Ignition - OFF
□ Master switch - OFF
□ Avionics master - OFF
□ Lights - ALL OFF
□ ACARS - COMPLETE flight log
□ ACARS - SUBMIT flight report
□ Documents - SECURE
□ Cockpit - SECURE
Checklist Usage Guidelines
  • All checklists must be completed in the specified order
  • Each item must be verified before proceeding to the next
  • Use the challenge-response method for multi-crew operations
  • Checklists may be abbreviated only in emergency situations
  • Any discrepancies discovered during checklist completion must be resolved before proceeding
  • Pilots are responsible for ensuring all items are completed
Note: These checklists are standardized for PartyBus VA operations. Aircraft-specific variations may apply. Consult the POH for aircraft-specific procedures.
