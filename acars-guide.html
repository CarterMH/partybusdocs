ACARS System Guide

The ACARS (Aircraft Communications Addressing and Reporting System) is used to track and log all flights.

Connection Settings
Server: acars.partybusva.com
Port: 6809
Protocol: FSPassengers
Update Interval: 30 seconds
Setup Instructions
  1. Download and install FSPassengers ACARS module
  2. Enter your pilot ID in the ACARS settings
  3. Configure the connection settings as shown above
  4. Test the connection before your first flight
  5. Start ACARS before loading your flight
Flight Logging

ACARS automatically tracks:

  • Flight duration
  • Distance flown
  • Fuel consumption
  • Landing rate
  • Flight path coordinates
Note: All flights must be logged via ACARS. Manual flight logs are not accepted.
Troubleshooting

If you experience connection issues:

  • Verify your pilot ID is correct
  • Check your internet connection
  • Ensure the server address is correct
  • Contact technical support if problems persist
