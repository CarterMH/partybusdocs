Getting Started Guide

This guide will help you get started with PartyBus Virtual Airline. Follow these steps in order to begin your virtual aviation career.

System Requirements

Before beginning, ensure your system meets the following requirements:

  • Flight Simulator: Microsoft Flight Simulator 2004, Flight Simulator X, or compatible simulator
  • Operating System: Windows XP, Windows Vista, or Windows 7
  • Internet Connection: Broadband connection required for ACARS and online features
  • Hardware: Minimum 1GB RAM, 2GB hard disk space
  • Graphics: DirectX 9.0 compatible video card
Initial Setup Procedure

Follow these steps to set up your system for PartyBus VA operations:

Step 1: Install Flight Simulator

Install your flight simulator software according to the manufacturer's instructions. Ensure all updates and service packs are installed.

Step 2: Download Aircraft

Download the PartyBus VA aircraft configurations from the Aircraft Configurations page. Extract the files to your flight simulator's aircraft directory.

Step 3: Install Paint Kit

Download and install the Paint Kit for PartyBus VA liveries. Follow the installation instructions included in the package.

Step 4: Configure ACARS

Set up the ACARS system according to the ACARS System Guide. This is essential for logging flights.

Step 5: Review Documentation

Read through the following essential documents:

  • Operations Manual
  • Pilot Handbook
  • Rules and Regulations
  • ACARS Guide
First Flight Checklist

Before your first flight, complete the following checklist:

  1. ✓ Flight simulator installed and updated
  2. ✓ PartyBus VA aircraft installed
  3. ✓ Paint kit and liveries installed
  4. ✓ ACARS system configured and tested
  5. ✓ Operations manual reviewed
  6. ✓ Rules and regulations read
  7. ✓ Orientation flight procedures reviewed
  8. ✓ Route database downloaded
Orientation Flight

All new pilots must complete an orientation flight. The orientation flight serves as a training exercise and ensures pilots are familiar with PartyBus VA procedures. See the Orientation Flight Procedures guide for detailed instructions.

Orientation Route: KSEA - KPDX (Seattle to Portland)
Distance: 129 nautical miles
Aircraft: Airbus A319, A320, or A321 (CEO variant)
Understanding the Documentation Structure

The PartyBus VA documentation is organized into several key sections:

  • Getting Started: This page - initial setup and orientation
  • Rules: Complete rules and regulations governing pilot conduct
  • Fleet: Detailed information about our aircraft fleet
  • Operations: Flight operations procedures and guidelines
  • Pilot Resources: Downloads, training materials, and reference documents
  • FAQ: Frequently asked questions and troubleshooting
Recommended Reading Order

For new pilots, we recommend reading the documentation in this order:

  1. Getting Started (this page)
  2. Rules and Regulations
  3. Pilot Handbook
  4. Operations Manual

After completing these, you may reference other documentation as needed.
