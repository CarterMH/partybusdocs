Getting Started Guide

This guide will help you get started with PartyBus Virtual Airline. Follow these steps in order to begin your virtual aviation career.

System Requirements

Before beginning, ensure your system meets the following requirements:

  • Flight Simulator: Microsoft Flight Simulator 2004, Flight Simulator X, or compatible simulator
  • Operating System: Windows XP, Windows Vista, or Windows 7
  • Internet Connection: Broadband connection required for ACARS and online features
  • Hardware: Minimum 1GB RAM, 2GB hard disk space
  • Graphics: DirectX 9.0 compatible video card
Initial Setup Procedure

Follow these steps to set up your system for PartyBus VA operations:

Step 1: Install Flight Simulator

Install your flight simulator software according to the manufacturer's instructions. Ensure all updates and service packs are installed.

Step 2: Download Aircraft

Download the PartyBus VA aircraft configurations from the Aircraft Configurations page. Extract the files to your flight simulator's aircraft directory.

Step 3: Install Paint Kit

Download and install the Paint Kit for PartyBus VA liveries. Follow the installation instructions included in the package.

Step 4: Configure ACARS

Set up the ACARS system according to the ACARS System Guide. This is essential for logging flights.

Step 5: Review Documentation

Read through the following essential documents in the order specified. All documents must be reviewed before beginning flight operations:

  1. Rules and Regulations - Must be read and understood completely. Contains all rules governing pilot conduct and operations.
  2. Code of Conduct - Establishes standards of behavior and professionalism. Must be reviewed and acknowledged.
  3. Flight Operations Manual - Complete flight operations procedures. Essential reading for all pilots.
  4. Standard Operating Procedures (SOP) - Detailed SOP covering all operational procedures. Must be reviewed thoroughly.
  5. Flight Checklists - All standardized checklists for flight operations. Must be understood before first flight.
  6. Pilot Operating Handbook (POH) - Aircraft specifications, limitations, and procedures. Essential reference document.
  7. Operations Manual - Additional operations information and procedures.
  8. Pilot Handbook - General pilot information and guidelines.
  9. ACARS System Guide - Complete ACARS setup and usage guide. Must be reviewed before first flight.
  10. Orientation Flight Procedures - Procedures for completing the required orientation flight.

Orientation Flight

All new pilots must complete an orientation flight. The orientation flight serves as a training exercise and ensures pilots are familiar with PartyBus VA procedures. See the Orientation Flight Procedures guide for detailed instructions.

Orientation Route: KSEA - KPDX (Seattle to Portland)
Distance: 129 nautical miles
Aircraft: Airbus A319, A320, or A321 (CEO variant)
Understanding the Documentation Structure

The PartyBus VA documentation is organized into several key sections, each serving a specific purpose in pilot training and operations:

  • Getting Started: This page contains initial setup procedures, system requirements, installation instructions, and orientation information for new pilots.
  • Rules and Regulations: Complete and detailed rules and regulations document governing all aspects of pilot conduct, flight operations, activity requirements, and disciplinary procedures. This document must be read and understood by all pilots.
  • Code of Conduct: Comprehensive code of conduct document establishing standards of behavior, professionalism, integrity, respect, safety, and accountability for all PartyBus VA members. This document defines expected conduct in all interactions and activities.
  • Fleet Information: Detailed specifications, performance data, capacities, limitations, and route compatibility information for all aircraft in the PartyBus VA fleet including A319, A320, and A321 CEO variants.
  • Flight Operations: Comprehensive flight operations manual covering all phases of flight operations including pre-flight procedures, departure procedures, en-route operations, arrival procedures, landing procedures, and post-flight procedures.
  • Checklists: Complete set of standardized flight checklists for all phases of flight including pre-flight inspection, cockpit pre-flight, engine start, before taxi, before takeoff, after takeoff, cruise, descent, before landing, after landing, and shutdown checklists.
  • Standard Operating Procedures (SOP): Detailed standard operating procedures document covering all aspects of flight operations including crew resource management, flight planning, engine start, taxi, takeoff, climb, cruise, descent, approach, landing, and post-flight procedures. This document also includes special procedures and record keeping requirements.
  • Pilot Operating Handbook (POH): Comprehensive pilot operating handbook containing aircraft specifications, limitations, performance data, weight and balance information, systems descriptions, normal procedures, emergency procedures, and handling information for A319/A320/A321 aircraft.
  • Pilot Resources: Central location for all downloadable resources including aircraft configurations, paint kits, operations manual, pilot handbook, route database, and training materials.
  • Contact Information: Department contact information, response times, communication guidelines, and office hours for various PartyBus VA departments.
  • FAQ: Comprehensive frequently asked questions document covering general questions, technical questions, flight operations questions, and rules questions.
Mandatory Reading Order

For new pilots, the following reading order is mandatory. All documents must be read before beginning flight operations:

  1. Getting Started (this page) - Initial setup and orientation information. Must be completed first.
  2. Rules and Regulations - Complete rules document. Must be read completely and understood. This is a mandatory document.
  3. Code of Conduct - Standards of behavior. Must be reviewed and acknowledged. This is a mandatory document.
  4. Standard Operating Procedures (SOP) - Operational procedures. Must be reviewed thoroughly. This is a mandatory document.
  5. Flight Checklists - All checklists. Must be reviewed and understood. Checklists are mandatory for all flights.
  6. Pilot Operating Handbook (POH) - Aircraft information. Essential reference. Must be reviewed before first flight.
  7. Flight Operations Manual - Operations procedures. Must be reviewed.
  8. Pilot Handbook - General pilot information. Should be reviewed.
  9. ACARS System Guide - ACARS setup and usage. Must be reviewed before first flight.
  10. Orientation Flight Procedures - Orientation flight information. Must be reviewed before orientation flight.
  11. Emergency Procedures - Emergency handling procedures. Should be reviewed.
  12. Radio Communications Guide - ATC communication procedures. Should be reviewed, especially if flying online.
Important: The Rules and Regulations, Code of Conduct, Standard Operating Procedures, and Flight Checklists documents are mandatory reading. These documents must be read and understood before beginning flight operations. Failure to be familiar with these documents may result in rule violations and disciplinary action.

After completing these, you may reference other documentation as needed.
