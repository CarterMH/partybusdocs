Emergency Procedures

In case of emergency situations during flight, follow these procedures.

Important: These are simulation procedures. In real aviation, emergency procedures are critical and must be memorized.
Engine Failure
  1. Maintain aircraft control
  2. Declare emergency to ATC: "Mayday, Mayday, Mayday"
  3. Follow engine failure checklist
  4. Identify nearest suitable airport
  5. Plan emergency landing approach
Loss of Communications
  1. Continue on assigned route
  2. Attempt to re-establish contact on emergency frequency (121.5 MHz)
  3. Follow last received clearance
  4. Land as soon as practical
Electrical Failure
  1. Switch to backup systems if available
  2. Notify ATC of reduced capability
  3. Request priority handling
  4. Plan for no-light landing if necessary
Fuel Emergency
  1. Declare fuel emergency to ATC
  2. Request vectors to nearest suitable airport
  3. Request priority handling
  4. Monitor fuel levels closely
Note: After any emergency landing, file a flight report and contact operations management.
