ACARS System

All flights must be logged using the ACARS system. The system tracks:

  • Flight duration
  • Distance flown
  • Fuel consumption
  • Landing rate
  • Flight path
ACARS Connection Settings:
Server: acars.partybusva.com
Port: 6809
Protocol: FSPassengers
Update Interval: 30 seconds

For detailed ACARS setup instructions, see the ACARS System Guide.

Flight Hours

Flight hours are automatically calculated and logged. Pilots can view their flight hours in their pilot profile.

Rank Minimum Hours
First Officer 0
Captain 50
Senior Captain 200
Line Check Pilot 500
Flight Procedures

All pilots must follow standard flight procedures:

  1. Pre-flight checks
  2. ATC communication (when online)
  3. Proper departure procedures
  4. En-route navigation
  5. Arrival and approach procedures
  6. Landing and post-flight checks
  7. ACARS submission

For detailed procedures, see the Operations Manual.
