PartyBus Virtual Airline Documentation

Welcome to the official PartyBus Virtual Airline documentation. This comprehensive documentation site contains all the information needed to operate as a pilot with PartyBus VA, including procedures, specifications, rules, and training materials.

Documentation Last Updated: March 15, 2008

Version: 1.0

Documentation Site: This site contains reference documentation only. All procedures, specifications, and guidelines are documented here for pilot reference.
Documentation Sections

The documentation is organized into the following sections. Use the navigation menu on the left to access each section:

Getting Started

Complete guide for new pilots including system requirements, initial setup procedures, first flight checklist, and orientation flight information. Essential reading for all new pilots.

Rules and Regulations

Complete rules and regulations governing pilot conduct, flight operations, activity requirements, and disciplinary procedures. All pilots must be familiar with these rules.

Fleet Information

Detailed specifications and information about the PartyBus VA fleet, including Airbus A319, A320, and A321 CEO variants. Includes performance data, capacities, and route compatibility information.

Flight Operations

Comprehensive flight operations manual covering pre-flight procedures, departure, en-route operations, arrival, landing, and post-flight procedures. Includes ACARS usage and flight hour requirements.

Checklists

Complete set of standardized flight checklists for all phases of flight including pre-flight inspection, engine start, taxi, takeoff, cruise, descent, approach, landing, and shutdown procedures.

Standard Operating Procedures (SOP)

Detailed standard operating procedures document covering all aspects of flight operations including crew resource management, flight planning, engine start, taxi, takeoff, climb, cruise, descent, approach, landing, and post-flight procedures.

Pilot Operating Handbook (POH)

Comprehensive pilot operating handbook containing aircraft specifications, limitations, performance data, weight and balance information, systems descriptions, normal procedures, and emergency procedures for A319/A320/A321 aircraft.

Code of Conduct

Complete code of conduct document establishing standards of behavior, professionalism, integrity, respect, safety, responsibility, and accountability for all PartyBus VA members.

Pilot Resources

Downloads, training materials, and reference documents including aircraft configurations, paint kits, operations manual, pilot handbook, route database, and training guides.

Contact Information

Department contact information, response times, communication guidelines, and office hours for various PartyBus VA departments including technical support, operations, and fleet management.

Frequently Asked Questions

Comprehensive FAQ covering general questions, technical questions, flight operations questions, and rules questions. Quick reference for common inquiries.
